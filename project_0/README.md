### Описание проекта    
Угадать загаданное компьютером число за минимальное число попыток.


### Какой кейс решаем?    
Нужно написать программу, которая угадывает число за минимальное число попыток

**Условия соревнования:**  
- Компьютер загадывает целое число от 0 до 100, и нам его нужно угадать. Под «угадать», подразумевается «написать программу, которая угадывает число».
- Алгоритм учитывает информацию о том, больше ли случайное число или меньше нужного нам.

**Метрика качества**     
Результаты оцениваются по среднему количеству попыток при 1000 повторений


### Краткая информация о данных

Алгоритм работает с целыми числами от 0 до 100.


### Этапы работы над проектом  

Был реализован двоичный поиск — классический алгоритм поиска элемента в отсортированном массиве (списке), использующий дробление массива на половины.


### Результат  

В среднем алгоритм угадывает число за 5 попыток.
